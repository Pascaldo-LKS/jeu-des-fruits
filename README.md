# 🍎 Jeu des Fruits

## 📱 Description
Jeu mobile développé avec Flutter.  
Le joueur doit identifier rapidement le bon fruit parmi plusieurs images affichées.

## Travail réalisé

Dans le cadre de ce projet, notre groupe a développé une application mobile intitulée Jeu des Fruits à l’aide du framework Flutter.

L’objectif principal du jeu est de permettre à l’utilisateur d’identifier rapidement un fruit spécifique parmi plusieurs fruits affichés à l’écran. Le joueur doit être rapide et attentif pour obtenir le meilleur score possible.

Les principales fonctionnalités réalisées sont :

Une interface d’accueil simple et intuitive
Un écran de jeu avec une grille de fruits interactive
Un système de score dynamique (+5 points pour une bonne réponse, -2 points pour une erreur)
Un système de chronomètre selon le niveau de difficulté
Une gestion des paramètres (activation/désactivation du son, choix du niveau : facile, moyen, difficile)
L’ajout de sons interactifs pour améliorer l’expérience utilisateur
Une interface responsive adaptée aux téléphones et aux écrans plus larges (PC)
L’intégration d’animations pour rendre le jeu plus dynamique
La sauvegarde du meilleur score
## Difficultés rencontrées

Au cours du développement de ce projet, nous avons rencontré plusieurs difficultés :

La gestion du temps (timer) et sa synchronisation avec les actions du joueur
L’intégration des sons dans l’application Flutter
L’adaptation de l’interface pour différents types d’écrans (responsive design)
La gestion de l’état global de l’application (score, paramètres)
La compréhension et l’utilisation de certains packages Flutter
La structuration du projet en plusieurs fichiers pour respecter une architecture propre

Ces difficultés nous ont permis d’améliorer nos compétences en développement Flutter et en organisation de projet.

## Préoccupations / limites

Malgré les fonctionnalités réalisées, certaines améliorations peuvent encore être apportées :

Ajouter plus de niveaux et de défis pour rendre le jeu plus intéressant
Améliorer le design graphique avec des animations plus avancées
Ajouter une base de données pour stocker les scores en ligne
Intégrer une musique de fond
Publier l’application sur le Play Store pour une utilisation réelle

Nous pensons que ces améliorations permettront de rendre l’application plus professionnelle et plus attractive pour les utilisateurs.
